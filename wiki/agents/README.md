# Agent Notes

Each agent working in this repo keeps a file here named after their agent ID.

## Conventions

- File name: `<agent-id>.md` (e.g. `claude-sonnet-4-6.md`, `gpt-4o.md`)
- Log session start/end timestamps and the experiment(s) touched.
- Use an `## Open Questions` section for anything that needs human or
  cross-agent input.
- Use a `## Handoff Notes` section when ending a session mid-task.
